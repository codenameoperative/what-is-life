DEB packages provide traditional Linux package management with automatic dependency resolution.

### Quick Install:
1. Download: `what-is-life_0.1.1_amd64.deb.zip`
2. Extract the zip file
3. Install: `sudo dpkg -i what-is-life_0.1.1_amd64.deb`
4. Run: `what-is-life` (from terminal) or find in applications menu

### Alternative Methods:

#### Command Line Installation:
```bash
# Navigate to download location
cd ~/Downloads

# Install the package
sudo dpkg -i what-is-life_0.1.1_amd64.deb

# If dependencies are missing, run:
sudo apt-get install -f

# Launch the game
what-is-life
```

#### GUI Installation:
1. Open your file manager (Nautilus, Nemo, etc.)
2. Navigate to the .deb file
3. Double-click the .deb file
4. Click "Install Package" in the software center
5. Search for "What is Life?" in your applications menu

#### System-wide Installation:
```bash
# Install for all users
sudo dpkg -i what-is-life_0.1.1_amd64.deb

# Update package database
sudo apt-get update

# Verify installation
dpkg -l | grep what-is-life
```
